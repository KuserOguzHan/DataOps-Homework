 # 1 Activate venv for airflow
 
```
[train@trainvm ~]$ source ~/venvairflow/bin/activate
```
 
 ```
(venvairflow) [train@trainvm ~]$ sudo systemctl restart airflow
 ```
  
  
 ```
(venvairflow) [train@trainvm ~]$ sudo systemctl start airflow-scheduler
 ```
